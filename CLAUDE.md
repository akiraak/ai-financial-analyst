# AI Financial Analyst

## プロジェクト概要
AIを活用した株価関連の財務分析ツール。株価データの取得・解析・可視化を行う。

## 分析対象
- 株価データ（日次・週次・月次）
- ファンダメンタルズ（PER、PBR、配当利回り等）
- 財務諸表（売上高、営業利益、純利益等）

## 技術スタック
- 言語: Python 3
- 進捗管理: TODO.md / DONE.md
