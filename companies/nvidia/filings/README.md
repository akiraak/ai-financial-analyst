## 会計年度（Fiscal Year）の定義

NVIDIAの会計年度は1月末締め。FY番号は期末の暦年と一致する。
- FY2023: 2022年2月〜2023年1月29日
- FY2024: 2023年1月30日〜2024年1月28日
- FY2025: 2024年1月29日〜2025年1月26日
- FY2026: 2025年1月27日〜2026年1月末（予定）
