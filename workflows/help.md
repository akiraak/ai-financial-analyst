# ヘルプ

## このツールについて

企業の決算資料から財務データを抽出し、チャート・解説付きの分析レポートページ（GitHub Pages）を生成するツールです。

## できること

1. **新しい企業を分析する** — 決算資料のダウンロードからデータ抽出・検証・レポート公開まで一連の流れを実施する
2. **既存の企業のデータを更新する** — 新しい四半期の決算資料を追加し、データ・レポートを最新化する

### 処理の流れ（共通）

いずれの場合も、以下のステップを順に実行します：

1. 決算資料のダウンロード（[download-filings.md](download-filings.md)）
2. 財務データの抽出・xlsx生成（[generate-financials.md](generate-financials.md)）
3. xlsx数値の検証（[validate-xlsx.md](validate-xlsx.md)）
4. 分析レポートページの生成（[analyze-and-visualize.md](analyze-and-visualize.md)）

## 「何をすればいい？」と聞かれたとき

1. **「新しい企業を分析する」か「既存の企業を更新する」かを聞く**
2. **分析対象の企業を聞く** — まだ企業が決まっていない場合、「どの企業を分析しますか？」と質問する
3. **分析期間を聞く** — 「過去何年分のデータで作成しますか？（例: 直近3年、5年、全期間）」と質問する
4. 選択に応じて、上記の処理の流れに沿って案内する

## 回答のルール

- 勝手に作業を始めない。必ずユーザーの選択を待つ
- 作業フローの途中であれば、次のステップを明示する
- 該当する作業フローがない場合は、ユーザーに何をしたいか聞く
